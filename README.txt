ESPAÑOL
------

Esto es una escena de demo en la que se usan triggers, físicas, animaciones, raycasts y audios.
Se tiene un inventario que en este caso es para la llave y se podría utilizar en un mapa más grande con
más objetos que se necesitasen para ciertas puertas u objetivos.

También se utilizan tags/names, y es necesario poner el carrito del principio en el Layer "Ignore raycast"
para poder coger la llave.

Hay que coger esta llave (con clic izquierdo) y luego se puede pasar por la puerta.
El movimiento se controla con WASD y linterna con espacio.

No hay fin del juego, pulsando Esc se puede salir (en el .exe, ya que hace un Application.Quit()).

------
ENGLISH
------
This is a demo scene in which we use triggers, physics, anims, raycasts and audios.
You have an inventory used for the key (in this case), but it could be used in a larger map with
more items needed for certain doors/objectives.

Besides, tags/names are used and it is necessary to set the first trolley's Layer to "Ignore raycast",
in order to be able to pick the key.

We must pick this key using LMB and then we can pass through the door.
Movement is controlled by WASD and lantern by Space.

There is no game ending, just press Esc to leave (if you're using .exe, it won't work on Unity Editor
since it calls an Application.Quit()).
